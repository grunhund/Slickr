To work Slickr needs PHP 5.
CHMOD 777 the following folders/files:
 /cache/
 /admin/install.php
 config.php
 disabled.htaccess

To install Slickr upload the content of Slickr0.5.rar to your FTP.
and then go to the file admin/install.php.
and follow the instructions :D
